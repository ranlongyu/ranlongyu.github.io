python 中一切皆对象，类也是一种对象，只不过这个对象（类）自身拥有创建对象（类实例）的能力。

type 是 python 的内建原类，type 对象本身又是 type 类的实例，可以用其动态的创建类：

在写一个类的时候为其添加 __metaclass__ 属性，用于指定原类。Python 会在类的定义中寻找 __metaclass__ 属性，如果找到了，Python 就会用它来创建类 Foo，如果没有找到，就会用内建的 type 来创建这个类。

在创建OBJ这个类的时候，__metaclass__ 起了作用，为 OBJ 增加了一个类属性 a。

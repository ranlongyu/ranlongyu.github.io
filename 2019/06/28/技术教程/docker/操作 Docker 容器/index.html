2. 后台运行

Ctrl+P Ctrl+Q 把交互式的容器转到后台，exit 退出交互式容器。

让 Docker 在后台运行而不是直接把执行命令的结果输出在当前宿主机下。此时，可以通过添加 -d 参数来实现。

此时容器会在后台运行并不会把输出的结果打印到宿主机上面(输出结果可以用 docker logs 查看)。

顺序容器

顺序容器有如下几种类型：

  1. ==vector== 可变大小数组
  2. ==deque== 双端队列
  3. ==list== 双向链表
  4. ==forward_list== 单向链表
  5. ==array== 固定大小数组，比内置数组类型更安全
  6. ==string== 与 vector 相似容器，专门保存字符

通常情况下，vector 是最好的选择。

容器操作可能使迭代器失效

向容器中添加或删除元素的操作可能使指向容器元素的指针、引用或迭代器失效。如果在一个循环中插入删除元素，不要缓存end返回的迭代器，每次都要重新调用。

vector 和 string 通常会分配比新空间需求更大的内存空间。vector 采取的策略似乎为每次需要分配内存空间时将当前容量翻倍。

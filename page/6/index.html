第三章 字符串、向量和数组

命名空间 using 声明

using namespace::name;

一旦有上述的语句，就可以直接访问命名空间中的名字，如：

using std::cin;
using std::cout; using std::endl;

注：头文件中不应该使用 using 声明。

标准库类型 string

使用 string 需包含：

#include <string> 
using std::string; // C++标准库

初始化

string s1;
string s2 = s1;
string s3 = "hello";
string s4(10,'c'); // s内容为十个c

读写 string 对象

string 对象从第一个正真字符开始读取，直到遇到下一个空格为止，如输入“ Hello World ”，实际存的是“Hello”。

输出时没有限制。

string s,s1,s2;
cin >> s;
cin >> s1 >> s2; // 多个输入输出可以连在一起
cout << s1 << s2 << endl;

使用 getline 输入整行，保留空格，遇到换行符结束：

string line;
getline(cin, line);

string 类型操作

字面值与 string 类型相加

string s2 = s1 + ",";    //正确，把一个string对象（s1）与一个字面值相加
string s3 = "hello" + "word";  //错误，两个运算对象都不是string,C++中的字符串字面值并不是标准库类型string的对象

用 for 循环处理字符串中每个字符

string str("some string");
for (auto c : str)      //对于字符串str中的每个c，执行某个操作，如果要修改str对象中字符的值，用引用，如把c改为&c
    cout << c <<endl;

用下标访问 string 对象单个字符

string s("some string")
if (!s.empty())
    s[0] = toupper(s[0]);   //只要字符串不是常量，就能修改下标运算符返回的值

标准库类型 vector

vector 表示对象的集合，其中所有对象的类型都相同，常被称为容器，需包含如下头文件

#include <vector>
using std::vector;

vector 初始化

最常见的方式是定义空 vector，然后当运行时获取元素的值逐一添加。

vector<int> v1;  //初始化v1为空
vector<int> v2(3);  //v2包涵3个初始值为0的int类型元素
vector<int> v3(3,2);   //v3包涵3个初始值为2的int类型元素
vector<int> v4{3,2};   //列表初始化，v4包涵2个int类型元素,值分别是3和2
vector<int> v5(v4);   //拷贝初始化，v5是v4的副本

使用函数 push_back 向 cector 对象添加元素

string word;
vector<string> text;
while (cin >> word){
    text.push_back(word);  //把word添加到text后面
}

vector 对象的其他操作和 string 类型很像，vector 对象（以及 string 对象）的下标只能用来访问已经存在的元素，而不能去添加元素！

迭代器

类似于指针功能，迭代器提供对对象的间接访问，相当于是下标。所有标准库容器都可以使用迭代器，但不一定支持下标。

迭代运算符

*iter      //返回迭代器所指元素的引用（注）
++iter     //令iter指示容器中的下一个元素
--iter     //令iter指示容器中的上一个元素
iter->mem  //解引用iter并获得该元素的名为mem的成员，等价于(*iter).mem

如

string s("some");
if (s.begin() != s.end()){      //确保s非空，注意s.end()指的是s尾元素的下一位置
    auto it = s.begin()；       //it指代s的第一个字符
    *it = toupper(*it);            //将s中的第一个字符改写为大写形式
}

任何改变 vector 对象容量的操作，比如 push_back 都会使该对象的迭代器失效。

数组

数组是类似于 vector 的数据结构，数组大小确定不变，没有 vector 灵活，不清楚元素个数，用 vector。

unsigned cnt = 42;
string bad[cnt];     //错误，cnt不是常量表达式（数组维度必须是一个常量表达式）。

字符数组的特殊形式，可以用字符串字面量进行初始化，注意结尾处有一个空字符。

char a1[] = "c++";  //结尾自动添加空字符，维度为4
char a2[] = {'c','+','+'};   //列表初始，维度为3

不能将数组的内容拷贝给其他数组作为初始值，也不能用数组为其他数组赋值

int a[] = {0,1,2};
int a2[] = a;     //错误
a2 = a;   //错误

数组其他用法于 vector 基本类似。

指针和数组

使用数组的下标时，编译器一般会把它转换成指针

string nums[] = {"one","two","three"};
string *p = &nums[0];    //p指向nums的第一个元素
string *p2 = nums;   //与上面等价
string *p3 = nums + 1;   //指针可以相加一个数，p3指向第二个元素

用函数 begin 和 end 访问数组，用其遍历数组较为方便，定义在标准库中 std::begin

string *p4 = begin(nums);    //指向首元素
string *p5 = end(nums);    //指向arr尾元素的下一位置

第二章 变量和基本类型

发表于 2017-01-02 | 更新于 2019-06-28 | 分类于 C++

第二章 变量和基本类型

执行浮点运算时选用double类型，因为float通常精度不够而计算代价相差无几。

当赋给带符号类型一个超过它表示范围的值时，结果是未定义的。

表达式中既有无符号数又有带符号类型时，那个带符号类型就会转换为无符号数进行计算。

编译器会在每个字符串的结尾处添加一个空字符（’\0’），因此，字符串字面值的实际长度要比它的内容多1。

初始化不是赋值，是创建变量时赋予一个初始值。

c++是一种静态类型语言，其含义是在编译阶段检查类型。

c++中大多数作用域都以花括号分隔，如for循环条件中定义的变量，只能在for循环的花括号中使用。

引用与指针

引用为对象起了一个别名，引用只能绑定它定义时的对象：

int ival = 1024;
int &refVal = ival; // refVal指向ival（是ival的另一个名字）
int &refVal2; // 错误，引用必须被初始化
int &refVal3 = 10; // 错误，引用的的初始值必须是一个对象

指针存放某对象的地址，要想获取地址，需要使用&：

int ival = 42;
int *p = &ival; // 指针p存放ival的地址
p = ival; // 错误，必须要先对变量取地址
int *p2 = 0，*p3 = nullptr; // 正确，定义了一个空指针

指针的类型一般情况下要和它所指向的对象严格匹配。把int变量直接赋值给指针是错误的，必须要先对变量取地址。

* 与 & 既能作为表达式里的运算符（解引用、取地址），又能作为声明里的一部分（指针、引用）。

对指针的引用：

int i = 42;
int *p;
int *&r = p; // r是对指针p的引用

const 限定符

如果想在多个文件之间共享const对象，必须在变量的定义之前添加extern关键字。

常量引用是对const的引用，与常量指针含义不同。

const int ci = 1024;
const int &r1 = ci; // 正确：引用及其对应的对象都是常量
int &r2 = ci; // 错误

常量引用仅对引用可参与的操作做出了限定，对于引用的对象本身是不是一个常量未作限定。因为对象也可能是个非常量，所以允许通过其他途径改变他的值：

int i = 42;
const int &r =i; // r绑定对象i，但是不允许通过r的途径修改i的值

指向常量的指针，指针的类型必须与其所指的对象类型一致，但是有两个例外，第一种是允许指向常量的指针指向一个非常量：

const double pi = 3.14;
const double *cptr = &pi; // 正确，指针的类型必须与其所指的对象类型一致
double dval = 3.14;
cptr = &dval; // 正确，指向常量的指针指向一个非常量，不能通过cptr改变dval的值

常量指针必须初始化，把*放在const关键字之前用以说明指针是一个常量，即不变的是指针本身的值而非指向的那个值：

int errNumb = 0;
int *const curErr = &errNumb; // curErr将一直指向errNumb
int ly = 1;
curErr = &ly; // 错误，curErr是常量指针，指针本身不能变
const double pi = 3.14156;
const double *const pip = &pi; // pip是一个指向常量对象的常量指针

顶层const表示指针是一个常量，底层const表示指针所指的对象是一个常量，是两个相互独立的问题。

处理类型

类型别名，是某种类型的同义词：

typedef double wages; // wages 是 double 的同义词
using SI = Sales_item; // SI是 Sales_item 的同义词

auto 类型说明符

auto 定义的变量必须有初始值

auto item = val1 + val2; // 相加的结果可以推断出 item 的类型
auto sz =0, pi = 3.14; // 错误，同一条语句中只能有一个数据类型，这里sz与pi类型不一致

decltype 类型指示符

decltype(f()) sum = x; // sum 的类型就是函数f的返回值

赋值是会产生引用的一类典型表达式，引用的类型就是左值的类型： 如果 i 是 int，则表达式 i=x 的类型是 int& 。

自定义数据结构

类通常被定义在头文件中，而且类所在的头文件名应与类的名字一样。

头文件通常包含哪些只能被定义一次的实体，如类、const和constrxpr变量。

确保头文件多次包涵仍能安全工作的技术是预处理器，当预处理器看到 #include 标记时就会用指定的头文件的内容代替 #include。

头文件保护符

#include 指令把一个名字设为预处理变量，预处理变量一般全部大写

#ifdef 判断给定变量是否已定义

#ifndef 判断给定变量是否未定义

#endif 用于结束一个 #ifdef 或 #ifndef 区域

头文件应该习惯性的设置保护符。

#ifndef SALES_DATA_H // 如果SALES_DATA_H没定义，则执行下面的语句到endif结束
#define SALES_DATA_H
#include <string>
sruct Sales_data{
    srd::string bookNo;
};
#endif
